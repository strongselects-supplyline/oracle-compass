## THE WEEKLY CADENCE (ADHD EXECUTION ENGINE)

Every week follows one of four modes. You don't choose — the calendar tells you.

### CORE RHYTHM (All Modes)
```
DEFAULT:    2 hours focused work → 20 minute walk/break → repeat
PUSH MODE:  3 hours focused work → 25 minute break → repeat (not daily — reserve for sprint days)
Breaks are WALKS, not scrolling. Movement resets the ADHD clock.
The time ranges in each mode below are containers. The 2hr/20min cycle runs inside them.
```

### MODE 1: PRODUCTION WEEK (batch recording/mixing/mastering)
```
Mon-Fri:  6:30-9 AM DoorDash (if needed for monthly target)
          9:30 AM S3 check-in → DAW open
          10 AM-12 PM deep production block 1 (one stage only: record OR mix OR master)
          12-12:20 PM walk break
          12:20-2:20 PM deep production block 2 (same stage)
          2:20-3 PM break + meal
          3-5 PM second production block (same stage, 2hr/20min cycle)
          5:30 PM done. Content review or meme posts if energy.
          8:30 PM wheels down.
Sat:      Overflow / Cyanite analysis / second draft listening
Sun:      Batch prep (meals). Grief journal. Rest. Kill List scan only.
```

### MODE 2: UPLOAD + PRE-RELEASE WEEK (7 days before drop)
```
Mon:      Upload to Amuse. Cover art finalized. Pre-save link generated.
Tue:      LYRC session (lyrc.studio): upload track + 2-3 trending video links → generate 30-50 lyric video variations. Cherry-pick top 10-15 for release cycle. Animated cover → Canvas snippet.
Wed:      Schedule best LYRC outputs across release week + post-release window. No manual CapCut meme edits needed — LYRC batch covers it.
Thu:      DM blitz round 1 (fans, playlist curators). Pre-save push on socials.
Fri:      Content batch: 3 more reels/shorts scheduled for release week.
Sat:      Rest + final listen.
Sun:      Grief journal. Pre-release hype post.
```

### MODE 3: RELEASE WEEK (drop day + 7 days after)
```
Release day: Post everywhere. Story blitz. Meme account drops 3-5 LYRC-generated edits (pre-batched during Mode 2 week).
Day 2-3:    DM blitz round 2 (new listeners from day-1 data). Playlist pitches.
Day 4-5:    Content Factory second wave (reaction-style, behind-the-scenes).
Day 6:      Pull first-week data. Assess save rate. Note for matrix.
Day 7:      Rest day.
Thu after:   Compliance — ASCAP, MLC, Songtrust, Musixmatch, final Cyanite for matrix.
```

### MODE 4: COMPOUND WEEK (no release active)
```
Mon-Wed:   Sync pitch prep (copy, one-sheets, mood descriptions from Cyanite data).
           Meme account content (pure memes, not music — maintain the ratio).
           Playlist outreach (independent curators, not editorial — those are release-week).
Thu-Fri:   Pre-production for next batch (beat selection, topline ideas, DAW prep).
           Live show booking / rehearsal if shows are upcoming.
Sat:       Open. DoorDash catch-up if behind on monthly target.
Sun:       Grief journal. Batch prep. Kill List scan.
```
