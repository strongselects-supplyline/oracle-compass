## SECTION 5: THE WAKING MIND PROTOCOL (new — Apr 8)

*Developed in Antigravity Session 3. Integrated into Oracle Compass Brain page and Kill List.*

### The 3-System Model

| System | Layer | What it does |
|--------|-------|-------------|
| **System 1** | Automatic | Fast, unconscious, habitual. Low ceiling. Drives you unless S3 is active. |
| **System 2** | Working Memory | Deliberate, effortful, fact-based. Most conscious thought. |
| **System 3** | Metacognitive | Symbols referring to your OWN mental states. Can reach into and rewire S1 & S2. No ceiling. |

S2 instructions ("Be calm") barely touch S1. S3 instructions ("I'm in a threat loop") reach S1 directly. Labeling IS the intervention.

### System 3 Activation Interrupts (Kill List tasks)

| Trigger | S3 Interrupt |
|---------|-------------|
| Feeling scattered | "What is my mind doing right now?" — wait for the answer, don't judge |
| Stuck in a session | Rate mental state 1–10. Rate again in 60 seconds. The act of rating breaks the loop. |
| About to react | Name the emotion: "I'm in a threat state." Naming activates regulation, not suppression. |
| Low energy spiral | Is this a body signal (interoception) or a thought loop? Different cause, different fix. |
| Distraction pull | "What does my distraction tell me about what I'm avoiding?" Curiosity, not judgment. |
