## 2. CONTENT & VISUAL

**Source:** `CONTENT_VISUAL_PLAYBOOK.md` → `lib/departments/content.ts`

**2026 Algorithm Reality (verified May 6, 2026):**
- Instagram: DM shares (sends-per-reach) weighted 3-5× over likes. 1 share ≈ 15 likes. Watch-through #2. Saves #3. Likes ≈ 0.
- TikTok: Completion rate → shares → sound re-use. 84% of Billboard Global 200 went viral on TikTok first.
- YouTube Shorts: Long-tail (performs months later). Second-largest search engine.

**Creation Order:** Create NATIVE on IG → export clean → upload TikTok → upload Shorts. Never reverse.

**Content Pillars (priority order):**
1. Talk to 'Em (3-4×/wk) — Direct-to-camera personality. Highest send rate.
2. Studio Sauce (2-3×/wk) — BTS production. Shows craft.
3. Release (8-day sprint per release) — Teaser → announce → drop → compound.
4. World Building (1-2×/wk) — Aesthetic reinforcement. 2 AM drive, Milwaukee, dark R&B.

**8-Day Sprint Template:** T-7 batch prep → T-5 teaser → T-3 world build → T-1 announce → T-0 drop (DM blitz) → T+1 momentum → T+3 compound → T+7 data + next.
