CapCut Template Build Specs

3 templates → ~52 assets
Template 1 — Quote-On-Screen
⭐ Highest ROI · Chaotic Good playbook · 500M+ views on the format
  • Project at 1080×1920
  • Import soft-focus ambient background video
  • Color grade to track palette
  • Audio: trim to hook (8–15s best moment)
  • Text: serif or minimal sans, white/off-white, slight shadow, centered
  • Animation: fade in 0:02 → hold → fade out at end
  • Export 1080p → Save as template → duplicate per track
  • Post from secondary / anon accounts only
  • No artist name, no "out now," no Spotify link
Template 2 — Late Drive / Night Aesthetic
Both SEE ME and ESL campaign kits use this
  • Film or source dashcam / window night footage
  • Apply 35mm grain overlay (CapCut filter or texture)
  • Apply track-specific color grade (ESL = warm amber)
  • Audio: start from vocal entry, skip instrumental intro
  • Text: small white, bottom-left or bottom-center
  • Text timing: invisible until 0:10 → fade in → hold to end
  • No artist name, no CTA
  • Export → Save as template
Template 3 — Bait-and-Switch / 4K Reveal
Jutsu #4 · Pattern interrupt · drives shares + rewatches
  • Record Part 1: iPhone front cam, casual bad lighting
  • Record Part 2: primary cam, studio lighting, Apollo monitoring
  • Place Part 1 → Place Part 2 → NO transition (hard cut only)
  • Part 1: add noise filter, lower saturation (intentionally degrade)
  • Part 2: full color grade, sharpen, track palette applied
  • Audio: lipsync in Part 1 → full studio audio in Part 2
  • End on strong vocal moment (sustain, riff, or silence)
  • Export → Save as template
