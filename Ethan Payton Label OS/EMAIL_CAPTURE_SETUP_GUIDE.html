Step-by-Step Setup

1

Create Your Account

  • Go to kit.com → click "Start for free"
  • Use your primary artist email (the one you check)
  • Artist name: Ethan Payton or past.El noir Records
  • Complete email verification
2

Verify Sender Settings

  • Go to Settings → Email
  • Set your "From Name": Ethan Payton
  • Set your "Reply-to": your real email
  • Add sender signature if prompted
  • This ensures your emails land in inboxes, not spam
3

Create the Landing Page

  • Navigate to: Grow → Landing Pages & Forms → Create New → Landing Page
  • Pick the cleanest/simplest dark template available (or any template — you'll customize it)
  • Set these fields:

Headline:

THE INNER CIRCLE

Subheadline:

First access to drops, unreleased tracks, and behind-the-scenes from past.El noir Records.

Form fields: First Name + Email (both visible)

Button text:

I'm In
  • Upload a photo (artist photo or EP cover art)
  • Click "Publish" when done
  • Copy the landing page URL — you'll need it for the next step
4

Set Up the Welcome Email (Incentive)

  • In your landing page settings → click Incentive
  • Enable "Send incentive email"
  • Subject line:
Welcome to the Inner Circle 🖤

Email body (copy/paste and edit):

Hey [First Name], You're in. I'm Ethan — I make music under past.El noir Records out of Denver. This list is where I drop things first: unreleased tracks, session footage, honest updates, and early access to everything before it hits socials. The ALL LOVE EP is out now — start here if you haven't heard it: → [Spotify link] Reply to this email and tell me your favorite track. I actually read these. — Ethan
5

Wire Into the Link-in-Bio Page

Once you have your Kit landing page URL:

  • Open LINK_IN_BIO.html in your Label OS folder
  • Find the handleEmailSubmit function in the script tag
  • Replace the placeholder redirect with:
// Option A: Redirect to Kit landing page window.location.href = 'YOUR_KIT_LANDING_PAGE_URL'; // Option B: Get the Kit embed form JS snippet // Kit dashboard → Landing Page → "Share" → "Embed" → copy the <script> tag // Paste it into LINK_IN_BIO.html above the closing </body> tag
💡 Recommendation Option B (embed) is better UX — the form stays on your page. Option A (redirect) is faster to set up. Start with A, upgrade to B later.
6

Add Kit Landing Page to IG Bio

  • Copy your Kit landing page URL
  • Go to Instagram → Edit Profile → Link → Add external link
  • Paste the Kit URL or the LINK_IN_BIO.html Vercel URL (once deployed)
  • Bio text suggestion:
Alt-R&B · Denver 🌑 ALL LOVE EP out now ↓
