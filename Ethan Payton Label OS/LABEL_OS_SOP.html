Mastering Chain — Exact Order, Every Time

  1. Parametric EQ — High-pass at 30–40 Hz (remove sub-rumble)
  2. M/S EQ — Hard cut SIDE channel below 120 Hz (mono the bass)
  3. G-Clip — 0% softness, placed BEFORE compressor (catches transient peaks)
  4. Multiband compression — gentle 2–3 dB per band, slow attack
  5. Multi-limiter sequence — 3–4 limiters in series, each doing 0.5–1 dB (NOT one limiter doing 4 dB)
  6. True Peak limiter — -0.3 dB ceiling, final stage only
  7. Target: -9 to -11 LUFS integrated (streaming) | -14 LUFS (sync/broadcast)
  8. DO NOT close FL Studio until all exports below are complete

Export Checklist (session OPEN)  sync-master-checklist

  • Master WAV — streaming (-9 to -11 LUFS, -0.3 TP)
  • Master WAV — sync (-14 LUFS, bypass final limiter, reset ceiling)
  • Instrumental — mute all vocal channels, bounce
  • Acapella — solo all vocal channels, bounce
  • Stem: Drums bus
  • Stem: Bass bus
  • Stem: Melody/Chords bus
  • Stem: Lead vocal
  • Stem: Background vocals
  • Verify: all files are 44.1kHz / 24-bit WAV minimum

Reference Check Protocol

  1. Studio monitors (your mixing position)
  2. Phone speaker (held at arm's length, not pressed to ear)
  3. AirPods
  4. Car (parked, engine off, moderate volume)
Vocal disappears on phone → boost mid-range. Bass wrong in car → adjust low-end balance. Harsh on AirPods → tame 2–5 kHz.

Cover Art — Photopea Workflow

  1. New document: 3000 × 3000 px, 300 DPI, RGB
  2. Background layer: gradient, texture, or photo base
  3. Subject layer: your photo isolated (Select Subject or pen tool)
  4. Typography: track/EP title + artist name. Must be readable at thumbnail size.
  5. Export: PNG (lossless) for Amuse. JPEG backup.
  6. Test: shrink to 300×300 in preview. Can you read the title? If not, increase font weight or contrast.

Spotify Canvas — CapCut Workflow

  1. Open cover art in Photopea, isolate elements onto separate layers → Content-Aware Fill gaps
  2. Export each layer as transparent PNG (File → Export As → PNG, transparency on)
  3. CapCut Desktop → New Project → 1080×1920 (9:16)
  4. Import all PNGs as separate tracks (stacked)
  5. Keyframe each layer: background slow zoom (105% over 8s), subject subtle pulse (100%→102%→100%), foreground faster than background
  6. Atmosphere overlay: dust/fog clip from Pexels → screen blend → 15–20% opacity
  7. Duration: 3–8 seconds. Must loop seamlessly (last frame = first frame)
  8. Export: MP4, 1080×1920, 30fps
